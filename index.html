Controller Friendly

Decktricks is designed entirely with controller use in mind. You can run complicated installers with just the click of a button, no keyboard/mouse/touchscreen required.

Game Mode Friendly: No More Desktop Mode

In an ideal world, Decktricks should be the only reason you ever need to load into Desktop Mode to install the most popular third-party software. And if you do need it, Decktricks has great support for Nested Desktop Mode, which is easier and faster to use.
