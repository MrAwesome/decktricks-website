What is Decktricks?

Decktricks is the best way to install 3rd-party software (like Decky, Discord, and Moonlight) from Game Mode on your Steam Deck.

What's special about Decktricks?

Designed entirely with controller use in mind. You can run complicated installers with just one button press - no keyboard, mouse, or touchscreen required.

Requires zero setup. Just run the installer and you're ready to go.

Updates itself, so you only need to install it once and you don't ever have to think about it again.

100% free and open-source.

(Technical Facts)

Open-source software, released under the Apache License 2.0.

Written almost entirely in Rust, thanks to the wonderful godot-rust project.

Written with extensibility in mind, with the ability to write custom modules for any arbitrary installation logic.

Completely GUI-independent CLI (and API), so all of the core functionality can be scripted or used in other projects.

Well-tested codebase, with robust and flexible CI and end-to-end testing setup.

Written by a former Big Tech SRE who has been gaming on Arch Linux since 2007.

No More Desktop Mode

For many users, installing Decktricks should be the last time you ever need to load into Desktop Mode. The most popular third-party programs are included by default, and Decktricks makes it easy to install/launch Nested Desktop Mode, which is much faster than the stock method.
